# Manual de Usuario - Control EMAP

**Dirigido a:** Todo el personal (Trabajadores y Jefes).

> [!abstract] ¿Qué es Control EMAP? Control EMAP es una aplicación móvil diseñada para hacer más fácil, seguro y transparente el proceso de salidas y misiones de trabajo. Permite pedir permisos desde el celular, autorizarlos al instante y asegurar que todo el personal esté protegido y ubicado durante su jornada laboral.

---

## 1. Guía para el Trabajador

Esta sección te explica cómo pedir un permiso de salida y cómo iniciar tu recorrido. La aplicación está diseñada para hacer el trabajo por ti, para que solo te preocupes por tu misión.

### 1.1 ¿Cómo pedir permiso para salir?

Cuando necesites salir de la oficina (la base) para hacer un trabajo, debes enviar una solicitud desde tu celular.

<img src="./imagenes/Pasted image 20260413225334.png" width="254"/>

**Pasos a seguir:**

1. Abre la aplicación y entra a la pantalla de solicitudes.
2. Escribe a dónde vas (**Destino**) y por qué necesitas salir (**Motivo**).
3. Presiona el botón de enviar.
4. Listo. Tu solicitud quedará en estado "Pendiente" hasta que el admin la revise.

---

### 1.2 ¿Cómo iniciar tu recorrido?

Una vez que los dos admins aprueben la solicitud, (basta que uno apruebe y pasen 5min para auto-aprobar la salida), la aplicación te avisará que ya puedes salir.

<img src="./imagenes/Pasted image 20260413230118.png" width="254"/>

**Pasos a seguir:**

1. Cuando veas que tu misión está "Aprobada", presiona el botón **Iniciar Recorrido**.
2. ¡Eso es todo! Puedes guardar tu celular, bloquear la pantalla o incluso abrir otras aplicaciones.
